**3. Cap the queue in PostScoreWithRetryAsync (STAB-01, D-02):**
Replace the `catch` block body in `PostScoreWithRetryAsync` so that when the retry fails, it checks the cap before enqueuing. Current catch block:
```csharp
catch (Exception ex)
{
    Debug.LogWarning($"[LeaderboardManager] Post failed, queuing retry: {ex.Message}");
    retryQueue.Enqueue((id, dist));
    StartRetryIfNeeded();
}
```
Replace with:
```csharp
catch (Exception ex)
{
    if (retryQueue.Count >= MaxRetryQueueSize)
    {
        Debug.LogWarning($"[LeaderboardManager] Retry queue full ({MaxRetryQueueSize}), dropping score {dist:F1} for {id}");
        OnLeaderboardSubmitFailed?.Invoke();
    }
    else
    {
        Debug.LogWarning($"[LeaderboardManager] Post failed, queuing retry: {ex.Message}");
        retryQueue.Enqueue((id, dist));
        StartRetryIfNeeded();
    }
}
```
